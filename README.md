## ダウンロード
[epcviewer v0.2.1 for macOS](https://github.com/mrhorin/epcviewer/releases/download/v0.2.1/epcviewer_v0.2.1_macos.zip)

## 特徴
- シンプルUI
- クロスプラットフォーム
- タブ表示機能
- タッチバー対応(macOS)

## 使い方
[epcyp](https://github.com/mrhorin/epcyp)などのYPブラウザにBBSブラウザとして登録して使用
